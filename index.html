    Summary of Docker Images¶

    Containers

    Jump off point for all docs of all container projects.

    Hugo

    For building static websites using Hugo.

    MkDocs

    For building static websites using MkDocs.

    Packer

    For building templated VMs using Hashicorp Packer.

    Packer Plus

    For building templated VMs using Ansible/Packer/etc.

    PowerCLI

    For running PowerShell and VMware PowerCLI scripts.

    Tanzu

    For interacting with Kubernetes/Tanzu clusters.

    Terraform

    For managing VMs/OVFs/etc on VMware vSphere.

    The Works

    All/Most of the above in a single image.
