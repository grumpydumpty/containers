# Container Image for containers on Photon OS

![](logo.png)

## Overview

Provides a base container image for building task-specific docker container images.

## Package Versions

This image includes the following components:

|                 |         |
|-----------------|---------|
| bash            | less    |
| bindutils       | mc      |
| ca-certificates | ncurses |
| coreutils       | openssh |
| curl            | shadow  |
| diffutils       | tar     |
| findutils       | tree    |
| gawk            | tmux    |
| git             | unzip   |
| htop            | vim     |
| jq              |         |
